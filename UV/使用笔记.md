安装脚本

```
irm https://astral.sh/uv/install.ps1 | iex
```

验证

```
uv --version
```

卸载 uv

```
Remove-Item "$env:CARGO_HOME\bin\uv.exe","$env:CARGO_HOME\bin\uvx.exe"
Remove-Item $env:LOCALAPPDATA\astral\uv -Recurse
```

PowerShell 永久配置

```
1、打开 PowerShell，输入打开配置文件
notepad $PROFILE


2、如果提示文件不存在，选「是」新建；
3、文件末尾粘贴这一行：
$env:Path = "C:\Users\WT\.local\bin;$env:Path"

配置文件路径，PowerShell 的用户启动脚本，每次打开 PowerShell 窗口会自动加载里面代码
D:\用户\文档\PowerShell\Microsoft.PowerShell_profile.ps1
```

初始化项目

```
uv init
```

新增开发工具到 dev 分组

如果后续要新增开发包，不要手动改 toml，用命令自动添加：

```
# 添加到全局dev分组，所有子项目共享
uv add --group dev ruff mypy pytest
```
