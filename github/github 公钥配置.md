# github 公钥配置

### **1、使用ssh-keygen命令生成ssh 密钥**

```
ssh-keygen -t rsa -C "你的邮箱"
```

说明：ssh-keygen命令可以生成rsa或dsa两种格式的密钥。在上面的示例中，使用-t rsa参数生成了id_rsa和id_rsa.pub两个文件，分别表示rsa私钥和rsa公钥。同理，可以使用-t  dsa参数生成dsa私钥和dsa公钥，生成的文件名分别是：id_dsa、id_dsa.pub。
