## 一、excel表合并&拆分

需求：

1. 根据某一分类（具体哪一列可由用户指定，填写一个数据即可），将Excel某个sheet表分拆成多个sheet
2. 根据行数，将Excel某个sheet表分拆成多个sheet
3. 将某个Excel多个sheet合并成一个sheet
   - sheet表头相同：合并后的数据共用一个表头
   - sheet表头不相同：合并后数据存在所有sheet表表头
4. 将多个Excel某个sheet表合并成一个Excel的一个sheet
   - sheet表头相同：合并后的数据共用一个表头
   - sheet表头不相同：合并后数据存在所有sheet表表头
