### 3、挂起禁止使用

挂起会出现黑屏，无法唤醒

```
systemctl mask sleep.target suspend.target hibernate.target hybrid-sleep.target

作用：彻底屏蔽所有睡眠 / 挂起目标，系统完全无法触发挂起
恢复命令：sudo systemctl unmask sleep.target suspend.target hibernate.target hybrid-sleep.target

+++++++++++++++++++++++++++++++
4 个东西就是 Linux 的4 种睡眠模式：
sleep.target → 轻度睡眠
suspend.target → 自动挂起
hibernate.target → 休眠
hybrid-sleep.target → 混合睡眠
```

### 6、虚拟机共享文件夹

```
sudo nano /etc/rc.local

写入下面内容
++++++++++++++++++++++++
#!/bin/sh
vmhgfs-fuse .host:/ /mnt/hgfs -o allow_other
exit 0
++++++++++++++++++++++++++

执行
sudo chmod +x /etc/rc.local
```
